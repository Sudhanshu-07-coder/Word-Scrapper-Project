# 📚 Books Web Scraper (Python)

A simple and efficient Python-based web scraper that extracts book titles and prices from the **Books to Scrape** website and stores the data in a CSV file.  
This project demonstrates practical web scraping using `requests` and `BeautifulSoup`.

---

## 🚀 Features

- Scrapes book **titles** and **prices**
- Handles multiple pages automatically
- Saves extracted data into a structured **CSV file**
- Clean, readable, and beginner-friendly Python code
- Uses widely adopted Python libraries

---

## 🛠️ Tech Stack

- **Python 3**
- **Requests** – for sending HTTP requests  
- **BeautifulSoup (bs4)** – for parsing HTML  
- **CSV module** – for data storage
